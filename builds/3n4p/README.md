# 3N4P outrunner

3 teeth, 4 magnets — fewest coils for a 3-phase hobby ESC. Plastic stator, outrunner layout.

| Version | Result | Notes |
|---------|--------|-------|
| [V1](v1/) | Failed | ~40 T, 24 AWG; BEMF ~mV |
| [V2](v2/) | **Spins** | Low speed only; hand-start; noisy; &lt;1 min run |

## Parts (BOM)

V1 and V2 use the **same purchased hardware** (magnets, bearings, wire, M3 kit, ESC). Only the printed parts differ.
